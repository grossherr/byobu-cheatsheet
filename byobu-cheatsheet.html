Byobu

Creation
  F2                               Create a new window
  Ctrl-a %                         Create a horizontal split
  Shift-F2
  Ctrl-a |                         Create a vertical split
  Ctrl-F2 
  Ctrl-Shift-F2                    Create a new session
Focus
  Alt-Left/Right                   Move focus among windows
  F3/F4
  Ctrl-a Ctrl-a                    Cycle focus among windows
  Ctrl-a <N>                       Focus window number N
  Ctrl-a "                         Select from list of all windows
  Alt-Up/Down                      Move focus among sessions
  Shift-Left/Right/Up/Down         Move focus among splits
  Shift-F3/F4
Movement
  Ctrl-F3/F4                       Move a split
  Ctrl-Shift-F3/F4                 Move a window
Resizing
  Shift-Alt-Left/Right/Up/Down     Resize a split
Renaming
  F8                               Rename the current window
  Ctrl-F8                          Rename the current session
Layout
  Shift-F8                         Toggle through split arrangements
  Alt-Shift-F8                     Restore a split-pane layout
  Ctrl-Shift-F8                    Save the current split-pane layout
  F11                              
  Alt-F11                          Expand split to a full window
  Shift-F11                        Zoom into a split, zoom out of a split
  Ctrl-F11                         Join window into a vertical split
Global
  Ctrl-F9                          Enter command and run in all windows
  Shift-F9                         Enter command and run in all splits
  Alt-F9                           Toggle sending keyboard input to all splits
Toggle
  Shift-F12                        Toggle on/off Byobu's keybindings
  Alt-F12                          Toggle on/off Byobu's mouse support
Detach
  F6                               Detach session and then logout
  Shift-F6                         Detach session and do not logout
  Alt-F6                           Detach all clients but yourself
Kill/Exit
  Ctrl-F6                          Kill split/window in focus
Scrollback
  F7                               Enter scrollback history
  Alt-PageUp/PageDown              Enter and move through scrollback
  Shift-F7                         Save history to $BYOBU_RUN_DIR/printscreen
Status
  F5                               Reload profile, refresh status
  Alt-F5                           Toggle UTF-8 support, refresh status
  Shift-F5                         Toggle through status lines
  Ctrl-F5                          Reconnect ssh/gpg/dbus sockets
  Ctrl-Shift-F5                    Change status bar's color randomly
Config
  F9                               Launch byobu-config window
Help 
  Shift-F1                         Display this help
Escape Sequence
  F12, Ctrl a                      Escape sequence
